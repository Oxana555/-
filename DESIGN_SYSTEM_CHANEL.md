# Chanel Design System (Source of Truth)

> Note: Brand naming has been migrated to **ViLab AI** while preserving this design system's premium visual language.

## Principles
- Quiet luxury aesthetic
- Spacious layouts
- Editorial typography hierarchy
- Soft-glass surfaces and subtle shadows
- High contrast for accessibility in light and dark appearances

## Foundations
- Colors are tokenized with semantic intent (`background`, `surface`, `textPrimary`, `accent`, etc.).
- Typography is tokenized by role (`display`, `title`, `body`, `caption`).
- Corner radii and spacing use a 4pt base grid.
- Components should prioritize legibility and tactile affordance.

## Core Components
- PrimaryButton
- SecondaryButton
- GlassCard
- TextField component
- PricingCard
- UploadCard
- ResultCard
- NavigationBar
- TabBar
- AlertBanner

## Screen Architecture
- Onboarding
- Home Dashboard
- Upload
- Results
- Pricing
- Profile
- Settings

## Motion
- Use spring transitions for card and banner interactions.
- Keep animation durations between 0.2s and 0.4s.

## Accessibility
- Minimum 44x44 tap targets
- Dynamic Type support
- Ensure semantic labels for upload/result status cards
